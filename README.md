# Semmelweis Handwashing Dashboard

This is a simple Streamlit app that visualizes Dr. Ignaz Semmelweis's
historical data on childbirth-related mortality in two clinics.

## Files

- `app.py` — main Streamlit application
- `yearly_deaths_by_clinic.csv` — dataset (place this in the same folder)
- `requirements.txt` — Python dependencies

## How to run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
